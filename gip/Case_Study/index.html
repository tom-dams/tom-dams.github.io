Case Study

Mijn onderwerp voor de case study was System Center Configuration Manager. Het is een redelijk bekend product en het wordt in veel ondernemingen gebruikt en bijvoorbeeld ook in onze school.

In het begin toen ik System Center Configuration Manager voor het eerst zag op mijn stagebedrijf dacht ik dat het heel moeilijk ging zijn, maar uiteindelijk na wat uitleg kon ik mijn weg er in terug vinden en was het toch niet zo moeilijk als verwacht.

Ik heb met System Center Configuration Manager veel task sequences moeten maken. Hieronder ziet u zo een task sequence. Deze worden gebruikt om ervoor te zorgen dat alle software en drivers op nieuwe computers komen te staan. Op mijn stagebedrijf heb ik zelf mijn task sequences moeten maken en hiermee nieuwe laptops die waren aangekomen klaar voor gebruik gemaakt.

GIP Tom Dams
